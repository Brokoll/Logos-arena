# PRD.md: Logos Arena MVP

## 1. 프로젝트 개요
- **목표**: AI 심판의 객관적인 판정을 통해 감정적인 소모전이 아닌 '논리적인 토론' 환경을 구축하고, 자신의 논리력을 증명하고 싶은 사용자를 유입시킨다.
- **핵심 가설**: "사용자는 자신의 주장이 AI에게 논리적으로 인정받을 때 더 높은 충성도를 보일 것이다."

## 2. 핵심 기능 (Features)
### 2.1 일일 아레나 (Single Topic Focus)
- 매일 00시에 관리자 혹은 시스템이 선정한 하나의 주제만 활성화된다.
- 사용자는 입장 시 '찬성' 또는 '반대' 진영을 반드시 선택해야 한다.

### 2.2 구조화된 주장 입력 (Logic-First Input)
- 일반적인 텍스트박스 대신 **[주장] - [근거] - [결론]**으로 분리된 입력 폼을 제공한다.
- 이는 사용자가 스스로 논리적 구조를 생각하게 만드는 장치로 작동한다.

### 2.3 AI 심판 시스템 (V1: Dummy & Wizard of Oz)
- 초기 버전은 OpenAI API를 호출하여 기본적인 논리 오류(인신공격, 순환 논리 등)를 체크하고 1~100점 사이의 점수를 부여한다.
- 실시간성이 중요하므로, 작성 즉시 '분석 중...' 애니메이션과 함께 결과를 노출한다.

### 2.4 쿨다운 및 랭킹
- 한 번의 게시글 작성 후 3분의 대기 시간을 부여한다.
- 논리 점수가 높은 유저는 '명예의 전당' 및 프로필에 랭킹이 표시된다.

## 3. 기술 스택
- **Framework**: Next.js 14 (App Router)
- **Database/Auth**: Supabase
- **Validation**: Zod
- **Styling**: Tailwind CSS (B&W Theme)
- **AI**: OpenAI API (Moderation & Analysis)
