# 🚨 에러 해결 방법 (필독!)

대장님! 방금 보신 **"저장 중 오류가 발생했습니다"** 에러는 **데이터베이스 보안 설정(RLS)** 때문입니다.
개발 중에는 누구나 쉽게 글을 쓸 수 있도록 제가 보안을 잠시 풀어주는 코드를 작성했습니다.

## ✅ 다음 단계를 실행해주세요!

1. **`supabase/schema.sql`** 파일을 엽니다. (제가 방금 내용을 업데이트했습니다!)
2. 파일의 모든 내용을 **복사(Ctrl+A, Ctrl+C)** 합니다.
3. **Supabase 대시보드**로 이동합니다.
4. 왼쪽 메뉴의 **SQL Editor**로 들어갑니다.
5. **New Query**를 누르고 복사한 내용을 붙여넣은 뒤 **Run** 버튼을 클릭합니다.

이 작업을 완료하시면, 웹사이트에서 정상적으로 글이 등록될 것입니다! 🚀
